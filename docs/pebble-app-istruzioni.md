# Istruzioni per creare e usare questa app Pebble

Questo file replica la checklist per lo scaffolding del progetto Pebble (prerequisiti, `pebble new-project`, struttura cartelle, build e install).

Adatta i passi alla tua macchina locale: clona questo repo, assicurati di avere il Pebble SDK 4 installato, poi usa `pebble build` e `pebble install` per provare l'app.
